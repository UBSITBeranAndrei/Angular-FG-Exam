# My Angular Application

## Project Description
This is a Single Page Application (SPA) built with Angular for the Applications Development 1 course. The project demonstrates proficiency in Angular fundamentals including component creation, routing, form handling, and TypeScript type declarations.

## Features
- Navigation Bar with routing links
- Home page with TypeScript primitive types demonstration
- User Profile page displaying custom TypeScript interfaces
- Registration Form with multiple input types and validation

## Technologies Used
- Node.js v22.13.0
- Angular CLI v17.x
- TypeScript v5.x
- Git

## Installation & Setup Instructions

### Prerequisites
- Node.js (v18.x or higher)
- Angular CLI
- Git

### Steps to Run Locally

1. Clone the repository:
```bash
git clone https://github.com/YOUR_USERNAME/my-angular-app.git
```

2. Navigate to the project directory:
```bash
cd my-angular-app
```

3. Install dependencies:
```bash
npm install
```

4. Run the development server:
```bash
ng serve
```

5. Open your browser and navigate to:
```
http://localhost:4200
```

## Project Structure
- `/navbar` - Navigation component
- `/home` - Home page component (default route)
- `/user-profile` - User profile display component
- `/registration-form` - Registration form component
